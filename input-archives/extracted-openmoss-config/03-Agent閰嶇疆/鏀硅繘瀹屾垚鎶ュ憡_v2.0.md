## 🛠️ 自动化工具

### 番茄数据自动化工具

**文件**: `~/OpenMOSS/app/tomato_auto_tool.py`

**功能**:
- ✅ 自动抓取番茄后台数据
- ✅ 每日自动生成数据报告
- ✅ 异常自动预警
- ✅ 竞品数据监控

**使用方法**:
```bash
# 生成每日报告
python3 ~/OpenMOSS/app/tomato_auto_tool.py --daily-report

# 实时监控（每小时）
python3 ~/OpenMOSS/app/tomato_auto_tool.py --monitor

# 竞品对比
python3 ~/OpenMOSS/app/tomato_auto_tool.py --competitors book1 book2 book3

# 查看帮助
python3 ~/OpenMOSS/app/tomato_auto_tool.py --help
```

```

    ├─ 规划师：世界观+主线
    ├─ 人物专家：角色设定
    └─ 数值专家：数值审核

阶段2：投票评审（1-2天）
    └─ 7Agent委员会，80分通过

阶段3：开篇优化（2-3天）
    ├─ 爽点设计师：黄金三章爽点设计
    ├─ Hook设计师：章末钩子设计
    └─ 情绪曲线设计师：情绪节奏设计

阶段4：日更运营（核心）
    ├─ 作家：日更6000字
```
